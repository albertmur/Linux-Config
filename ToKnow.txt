1. To make dolphin work, breeze-icon-themes and qt6ct (or newer versions) must be installed. Then, open the /etc/environment file and add to the bottom QT_QPA_PLATFORMTHEME="qt5ct". Finally, with rofi, open up qt5 settings and in the top bar select the adwaita theme. Don't forget to check the icon tab and select the icon theme.  

2. You can set the right resolution of grub in etc/default/grub with  GRUB_GFXMODE="1920x1080" and after running grub2-mkconfig -o /boot/grub2/grub.cfg.

3. To install EWW on OpenSuse (the widget system), besides following the official instructions, the packages that have to be installed are:
sudo zypper in gtk3-devel gtk-layer-shell-devel pango-devel gdk-pixbuf-devel libdbusmenu-gtk3-devel cairo-devel glib2-devel gcc glibc

4. To use the default font of VSCode, install the saja-cascadia-code-fonts.
